# defender
Defender Enable/Disable Tool

# basic
vs2022 + c# + net3.5

# usage(64-bit only)
<pre>
defender.exe <-s|-r> [-d directory] [-n name]
-s        Stop Defender
-r        Restart Defender
-d        Install wsc to the directory, default is C:\Program Files\Avast Software\Avast
-n        Antivirus product name, default: Avast Antivirus
</pre>
